# Contributing to Spam Mail Classifier

First off, thank you for considering contributing to this project!

## How Can I Contribute?
- **Reporting Bugs:** Create an issue describing the bug and how to reproduce it.
- **Suggesting Enhancements:** Open an issue to discuss new features or improvements.
- **Pull Requests:** 
    1. Fork the repository.
    2. Create a new branch for your feature.
    3. Commit your changes with clear messages.
    4. Push to the branch and open a Pull Request.

## Coding Standards
- Maintain clean and readable Python code.
- Ensure all dependencies are updated in `requirements.txt`.
- Document any new features in the `README.md`.
